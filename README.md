## Use as Extension

This repository can be added as an **extension** in MakeCode.

* open [https://arcade.makecode.com/](https://arcade.makecode.com/)
* click on **New Project**
* click on **Extensions** under the gearwheel menu
* search for **https://github.com/sabrinacohen/marvelous-maggie-mysteries-2** and import

## Edit this project ![Build status badge](https://github.com/sabrinacohen/marvelous-maggie-mysteries-2/workflows/MakeCode/badge.svg)
